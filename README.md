# Issue Tracker

A clean, responsive, and dependency-free Kanban-style issue tracker built with vanilla HTML, CSS, and JavaScript.

## Features

- **Three-State Kanban Board**: Manage tasks across "To Do", "In Progress", and "Completed" states.
- **Native Drag and Drop**: Intuitively move issues between columns using the standard HTML5 Drag and Drop API.
- **Local Storage Persistence**: Your issues are saved locally in the browser, ensuring data is not lost upon page refresh.
- **Clean Architecture**: Built without frameworks to demonstrate proper modular vanilla JavaScript structures.
- **Modern UI**: Lightweight visual styling prioritizing user experience and clarity.

## Getting Started

Because this application relies solely on standard web technologies, there are no build steps or dependencies to install.

### Running Locally

1. Open the project folder.
2. Double-click the `index.html` file to open it in your preferred web browser.
3. Alternatively, you can run a local development server (e.g., using Python, Node, or an IDE extension like Live Server) if you plan on making modifications and want automatic reloads.
   - Using npx: `npx serve .`
   - Using Python: `python -m http.server 8000`
